# Parameterized Synchronous FIFO Design and Verification

This project implements and verifies a parameterized single-clock synchronous FIFO in SystemVerilog.

## Planned Features

- Parameterized data width
- Parameterized depth
- Single-clock synchronous FIFO RTL
- Directed tests
- Self-checking scoreboard
- Randomized tests
- Assertions
- Functional coverage or coverage plan
- Waveform debugging
- Regression script

## Tools

- WSL Ubuntu
- Icarus Verilog
- GTKWave
- Verilator
- Git/GitHub
